### Simple game about exploring hotel and talking with various NPCs inspired by Wuppo

## How to play

### Play in browser here :
https://jinrty.itch.io/hotel-talk

### Download Godot 
a) Download all files
b) install Godot (4.x)
c) import and press F5

 ## What was used

1. Godot 4.3
2. Aseprite and Libresprite
3. Sour gummy font
